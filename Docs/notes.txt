Section 5: Introduction to CSS
	CSS = Cascading Style Sheets
	
	Three Ways to Add CSS:
		Inline:
			<tag style="css"/>
		Internal:
			<style>css</style>
		External:
			<link href="style.css"/>
			
	CSS Selectors: Specifies which part of the document to appply the css, h1 in this example
		Element Selector:
			h1 {
				color:blue
			}
		Class Selector: (Add 'class="red-heading"' to the desired html element
			.red-heading {
				color:red
			}
		ID Selector: (Add 'id="blue-heading"' on the desired html element)
			#blue-heading {
				color:blue
			}
		Use Class when multiple elements will share the same CSS, use ID when only 1 element will have it.
		
		Attribute Selector:
			p[draggable]{
				color:red
			}
		Universal Selector: Applies to the whole document
			* { 
				color:black
			}
Section 6: CSS Properties
	Font Properties:

			1rem: 100% of root
		font-weight:
			normal bold
			lighter bolder
			number (100-900)
		font-family:
			Multiple families here
			Most Common: 
				Helvetica, sans-serif
				Times New Roman, serif
			fonts.google.com 
		text-align:
			center,left, right, start, end
	CSS Box Model (Margin, Padding & Border)
		border: thickness style color,ie 
		border: 10px solid black;
		
		padding: pushes the border of the element by a given space, ie:
		padding: 20px;
		
		margin: (OUTSIDE THE BOX)
		margin: creates a distance between the element and all others by a given quantity, ie:
		margin: 20px;

Section 7: Intermediate CSS
	The Cascade Hierarchy:
		Position (Bottomless if same property), for example:
			li {
				color: red;
				color: blue;  <--- This will be the final value.
			}
		Specificity (id beats class)
			.li {color: red;}
			#first-id {color: blue;} <--- This will be the final value
		
		Type (external css, then internal css, finally inline css)
			<link rel="stylesheet" href="./style.css"> <--- This will style first
			<style> </style>  <--- This will style on top of previous one
			<h1 style=" "> Hello </h1> <-- This will style on top of all others
		
		Importance
			li{
				color: red;
				color: green !important; <--- This will be the final value.
			} 
	Combining Selectors:
		We can combine class, element, and id selectors for more specified css.
		in the case of a:
			<div class="box">
				<p class="done"> Paragraph. </p>
			</div>
			
		Categories: *Use combineCSS/index.html for better understanding*
			Group: h1, h2
			Child: .box > p
			Descendent: .box li
			Chained: li.done
			Multiple Combiners: ul p.done
	CSS Position:
		Static (Literally No Changes)
		Relative (Position relative to default position)
		Absolute (Position relative to nearest relative positioned ancestor or top
			left corner) *TOP LEFT IF IT DOESNT HAVE A RELATIVE ANCESTOR*
		Z-index (Displaying layers) 
			*If we give an element absolute position it takes the element to
			another layer.*
		Fixed (Position relative to top left corner of browser)

		TIP: To Display Content Of Top of Another:
			Background content: position: relative
			Top Content:	position: absolute
Section 8: Advanced CSS
	CSS Display (Inline, Block, inline-block)
		Inline: Displays everything on the same line. (Width & Height will be default to content size)
		Block: Takes the entire full width. (Second element will be displayed below it)
		Inline-Block: Same as inline but you can customize the height & width.
		None: Hides the element from the webpage.
